1) What is the definition of server port?
The answer is: a server is an endpoint that receives a request and a port 
is used to identify the "location" of the  requested endpoint. Therefore a 
server port is a specific server.

2) What is server port range?
The answer is: 0 to 65536

3) How many bits does number above represent?
The anwer is: 16-bits

4) State 10 common server ports
The answer is: 20, 21, 22, 25, 53, 80, 123, 500, 443, 3389
